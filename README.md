# Stochastic-Methods-in-Finance
This group project was realised in the context of the "Stochastic Methods in Finance" at the University of St. Gallen.

# Warning 
Instead of using Monte Carlo simulation, we "brute-forced" the process by simulating all the possible paths (~33 millions). You can expect your computer to run for 5-10 minutes when running sim.py, and be aware that the output data is heavy (~1.05GB)
